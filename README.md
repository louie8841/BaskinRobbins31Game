# BaskinRobbins31Game

## What it is
- Baskin Robbins 31 game made with python tkinter

## How to play Baskin Robbins 31 game
1. Decide the order
2. Players count sequentially by order(A player cansay 1 or 2 or 3 numbers in one time)
3. the person who said 31 loose
